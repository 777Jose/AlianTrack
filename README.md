# ALIANTRACK - Web corporativa

Proyecto base en React + TypeScript + Vite + Tailwind CSS

## Tecnologías incluidas
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Hook Form
- Radix UI (menú móvil)

## Cómo ejecutar
```bash
npm install
npm run dev
```

## Estructura
- `src/app/components`: componentes por sección
- `src/lib/data.ts`: textos y contenido editable
- `src/styles/index.css`: utilidades globales y clases compartidas
